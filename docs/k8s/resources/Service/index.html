Service

Provides a Kubernetes Service

Examples#

Create a NodePort service for a Deployment#

This example creates a NodePort service for use with a Deployment:

const appLabel = "myLabel";
const namespace = provider.makeNamespace({
name: "myNamespace",
});
const service = provider.makeService({
name: "myService",
dependencies: { namespace },
properties: () => ({
spec: {
selector: {
app: appLabel,
},
type: "NodePort",
ports: [
{
protocol: "TCP",
port: 80,
targetPort: 8080,
},
],
},
}),
});

Create a Headless Service for a StatefulSet#

This example creates a Headless service for use with a StatefulSet:

const appLabel = "db";
const namespace = provider.makeNamespace({
name: "myNamespace",
});
const service = provider.makeService({
name: "postgres-service",
dependencies: { namespace },
properties: () => ({
spec: {
selector: {
app: appLabel,
},
clusterIP: "None", // Headless service
ports: [
{
protocol: "TCP",
port: "5432",
targetPort: "5432",
},
],
},
}),
});
