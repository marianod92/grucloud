Backend Bucket

Provides a backend bucket.

Code​

const bucketName = "myfancybucketname";

provider.storage.makeBucket({
  name: bucketName,
});

provider.compute.makeBackendBucket({
  name: "backend-bucket",
  properties: () => ({
    bucketName,
  }),
});
