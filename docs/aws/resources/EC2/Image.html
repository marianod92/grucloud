AMI

Provides an Amazon Managed Image.

Example​

EC2 running on Ubuntu 20.04​

  • Find the Ubuntu Server 20.04 LTS image running on x86_64.
  • Provides the image as an EC2 dependency.
const image = provider.EC2.useImage({
  name: "ubuntu 20.04",
  properties: () => ({
    Filters: [
      {
        Name: "architecture",
        Values: ["x86_64"],
      },
      {
        Name: "description",
        Values: ["Canonical, Ubuntu, 20.04 LTS, amd64 focal*"],
      },
    ],
  }),
});

const server = provider.EC2.makeInstance({
  name: "myserver",
  dependencies: {
    image,
  },
  properties: () => ({
    InstanceType: "t2.micro",
  }),
});

Amazon Linux 2​

Here is the Amazone Linux 2 image:

const image = provider.EC2.useImage({
  name: "Amazon Linux 2",
  properties: () => ({
    Filters: [
      {
        Name: "architecture",
        Values: ["x86_64"],
      },
      {
        Name: "description",
        Values: ["Amazon Linux 2 AMI *"],
      },
    ],
  }),
});
