Resources List

List of resources for provider aws:

  • APIGateway: DomainName, Account, ApiKey, RestApi, Stage, Authorizer
  • ApiGatewayV2: DomainName, Api, Stage, Authorizer, ApiMapping, Integration, Route, Deployment
  • AppSync: GraphqlApi, DataSource, Resolver
  • AutoScaling: AutoScalingGroup, AutoScalingAttachment, LaunchConfiguration
  • ACM: Certificate
  • CloudFront: Distribution
  • CloudWatchEvents: EventBus, Rule
  • CloudWatchLogs: LogGroup
  • CognitoIdentityServiceProvider: UserPool, IdentityProvider
  • DynamoDB: Table
  • EC2: KeyPair, Image, Volume, Vpc, InternetGateway, NatGateway, Subnet, RouteTable, RouteTableAssociation, Route, SecurityGroup, SecurityGroupRuleIngress, SecurityGroupRuleEgress, ElasticIpAddress, Instance, LaunchTemplate, NetworkAcl
  • ECR: Repository, Registry
  • ECS: CapacityProvider, Cluster, TaskDefinition, Service, TaskSet, Task, ContainerInstance
  • EKS: Cluster, NodeGroup
  • ELBv2: LoadBalancer, TargetGroup, Listener, Rule
  • IAM: OpenIDConnectProvider, User, Group, Role, Policy, InstanceProfile
  • KMS: Key
  • Lambda: Layer, Function, EventSourceMapping
  • RDS: DBSubnetGroup, DBCluster, DBInstance
  • Route53: HostedZone, Record
  • Route53Domains: Domain
  • S3: Bucket, Object
  • SQS: Queue
  • SSM: Parameter
